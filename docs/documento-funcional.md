# Documento funcional - MotoTracker

MotoTracker es una aplicación móvil para usuarios de motos. Permite registrar una o varias motos y llevar un control básico del kilometraje, gastos, mantenimientos y viajes programados.

## Objetivo

Centralizar la información principal de la moto para que el usuario pueda organizar gastos, controlar el estado de mantenimiento y planificar viajes.

## Tecnologías

- React Native + Expo
- Java + Spring Boot
- MySQL

## Vistas principales

- Login
- Inicio
- Gastos
- Estado
- Viajes
- Configuración
